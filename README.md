## Space Something - Asteroids RPG game

![Space Something](screenshot.png "$(Space Something)")

### Description

RPG game prototype made for Raylib Slo-Jam 2023 in C++20.

### Controls

Keyboard:
    - Arrows to move
    - Spacebar to shoot and interact
